简短发布介绍：

```text
我做了 Link Shelf，这是一个小型 Windows 工具。它可以把选中的文件或目录移动到可携带缓存根目录，并在原路径创建符号链接。它适合同步开发工具配置、备份分散的应用状态，或者在新机器上恢复链接。Syncthing 和它很搭，但任何备份或同步工作流都可以使用同样思路。
```
