简短发布介绍：

```text
我做了 Link Shelf，这是一个 Windows 上的便携应用状态收纳工具。它可以把分散的应用状态和开发工具设置收进一个可同步缓存根目录，同时通过符号链接让应用继续使用原路径。它支持推荐本机常见配置路径、在新机器上恢复链接、用命令行检查链接健康，也可以在不想继续管理时把项目搬回原位。Syncthing 和它很搭，但任何备份或同步工作流都可以使用同样思路。
```
